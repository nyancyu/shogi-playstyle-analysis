# 棋士プレースタイル分類

将棋棋士の棋譜データを機械学習で分析し、プレースタイルを自動分類するプロジェクトです。

## プロジェクト概要

プロ棋士5名（羽生善治・藤井聡太・伊藤匠・菅井竜也・渡辺明）の棋譜データを収集し、
勝率・平均手数・戦型などの特徴量を抽出。k-meansクラスタリングとPCAを用いてプレースタイルを可視化しました。

## 使用技術

- Python 3.11
- cshogi：KIF形式の棋譜パース
- pandas / numpy：データ処理
- scikit-learn：StandardScaler・KMeans・PCA
- matplotlib：可視化

## データ

- 対象棋士：羽生善治・藤井聡太・伊藤匠・菅井竜也・渡辺明
- 各棋士50局・計250局のKIF形式棋譜
- 将棋DB2より取得

## 特徴量

| 特徴量 | 説明 |
|--------|------|
| 勝率 | 総合勝率 |
| 平均手数 | 1局あたりの平均手数 |
| 先手勝率 | 先手番の勝率 |
| 後手勝率 | 後手番の勝率 |
| 短期決戦率 | 100手未満で終局した割合 |
| 長期戦率 | 130手超で終局した割合 |
| 勝利時平均手数 | 勝ったときの平均手数 |
| 敗北時平均手数 | 負けたときの平均手数 |
| 振り飛車率 | 振り飛車戦型の割合 |

## 分析手法

1. KIF形式の棋譜ファイルをcshogiでパース
2. 棋士名の正規化（段位・タイトル・表記揺れを除去）
3. 投了手数から勝者を判定
4. 9つの特徴量を集計
5. StandardScalerで標準化
6. k-means（k=3）でクラスタリング
7. PCAで2次元に圧縮して可視化

## 結果

![棋士プレースタイル分類](images/playstyle.png)

- **菅井竜也**：振り飛車率0.90・平均手数120手と長期戦型で独立したクラスタを形成
- **伊藤匠**：勝率0.74・先手後手ともに安定した高勝率で単独クラスタ
- **藤井聡太・羽生善治・渡辺明**：居飛車型のオールラウンドクラスタ

PC1（寄与率62.6%）が振り飛車・長期戦を表す軸として最も重要な特徴量となった。

## 実行方法

cd shogi-playstyle-analysis

pip install cshogi pandas numpy scikit-learn matplotlib jupyter

jupyter notebook

## 注意事項

棋譜データは著作権の関係でリポジトリに含めていません。
将棋DB2より各棋士の棋譜をKIF形式でダウンロードしてください。
